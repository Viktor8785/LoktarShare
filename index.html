Вопросы и ответы

  • вопрос №1

    Сервис работает с агентскими кабинетами?

    — Да. Сервис работает с любыми кабинетами, к которым у вас есть доступ

  • вопрос №2

    С какими форматами работает сервис?

    — Сервис работает с любыми форматами, которые доступны при таргетинге

  • вопрос №3

    Есть ли возможность копировать из кабинета в кабинет?

    — Конечно. Причем, при копировании все данные переносятся корректно
